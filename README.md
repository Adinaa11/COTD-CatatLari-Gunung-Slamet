## 📖 Deskripsi Aplikasi

CatatLari adalah aplikasi Android yang digunakan untuk mencatat aktivitas lari pengguna.  
Aplikasi ini memungkinkan pengguna untuk menyimpan data seperti tanggal, jarak tempuh, durasi, serta catatan tambahan.

Data disimpan secara lokal menggunakan database sehingga pengguna dapat melihat, mengedit, dan menghapus riwayat aktivitas lari kapan saja.

---

## ✨ Fitur Aplikasi

### 🔐 Autentikasi User
- Login menggunakan email dan password
- Register akun baru

### 📋 Menampilkan Data Lari (Read)
- Menampilkan daftar aktivitas lari

### ➕ Menambahkan Data Lari (Create)
- Menambahkan data tanggal, jarak, durasi, dan catatan

### ✏️ Mengedit Data Lari (Update)
- Mengubah data yang sudah ada

### 🗑️ Menghapus Data Lari (Delete)
- Menghapus data dari daftar

## 🎯 Tujuan Aplikasi

Aplikasi ini dibuat untuk:
- Membantu pengguna mencatat aktivitas olahraga lari
- Menyimpan riwayat aktivitas secara terstruktur
- Memudahkan pengguna dalam memantau aktivitas lari
- Menerapkan konsep pengembangan aplikasi Android seperti MVVM dan Room Database
